Weather App

Initially I added a unit conversion for celsius, but ultimately opted out of that to simplify my code. This app is also fully responsive on mobile devices.

Dictionary App

I built this responsive dictionary app with React.js using a Dictionary and Image API. The user can search for any word and receive the definition, pronunciation, synonyms, and images associated with that word. I used an earthy muted pastel color palette to give the app a soft vintage feel, almost like opening up your local coffee shop's dictionary.
